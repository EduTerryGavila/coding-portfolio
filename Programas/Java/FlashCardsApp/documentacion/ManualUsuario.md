# Manual De Usuario

## Inicio/Registro De Sesión:

Para utilizar la aplicación debes de estar registrado. Para ello debes de introducir tu nombre, correo, contraseña y si vas a usar la cuenta como creador o estudiante. Una vez registrado hay que iniciar sesión con tu correo y contraseña eligiendo si quieres iniciar sesion como creador o estudiante.

- [Ver ventana de registro](../diseño/ventanas/Registro.jpg)

- [Ver ventana de inicio de sesion](../diseño/ventanas/Login.jpg)

## Menú Principal:

Una vez hecho lo anterior, aparecerá el menú principal correspondiente. Si inicias como estudiante tendras opciones para ver tus estadisticas, buscar cursos y cursarlos.

- [Ver ventana menu estudiante](../diseño/ventanas/MenuPrincipalEstudiante.jpg)

Si inicias como creador podrás ver tus estadisticas, buscar cursos para descargarlos, ver tus cursos para editarlos o borrarlos y por ultimo crear cursos.

- [Ver ventana menu creador](../diseño/ventanas/MenuPrincipalCreador.jpg)

## Crear Curso:

Debes pulsar esta opción si quieres crear un curso nuevo de cero. Una vez pulsado, tendrás que elegir el nombre del curso a crear. Para añadir las preguntas del curso se debe completar la informacion de la pregunta y darle al botón de añadir pregunta. Podrás elegir entre los siguientes tipos de pregunta:

- Verdadero/falso: Habrá que poner en el hueco correspondiente una afirmación la cual puede ser verdadera o falsa. Por último, se debe de seleccionar la respuesta correcta es decir, o verdadero o falso. Ejemplo: Creo la afirmación "2 + 2 = 5" y selecciono la opción "Falsa" de manera que quien realice el curso solo obtendrá la pregunta por buena si selecciona la opción "Falso" dentro del test.

- Completar hueco: En este tipo de pregunta se debe de poner en el lugar correspondiente una frase con un hueco (el hueco puede ser una palabra, una letra, etc) el cual debe de ser completado por la persona que realiza el curso. Para ello, se obtendrán 3 opciones de las cuales habrá solo una correcta. Ejemplo: Creo la afirmación "2 + _ = 4" y en "palabra a completar" pongo "2". En opciones incorrectas pongo por ejemplo "3" y "1". Solo se dará por buena si el estudiante que realice el curso pulsa "4".

- Pregunta abierta: Simplemente se debe de seleccionar una pregunta y su correspondiente respuesta. Ejemplo: creo la pregunta "¿Cuánto es 2 + 2?" y le pongo de respuesta "4". El estudiante deberá introducir "4" por teclado cuando le salga esta pregunta para que se le se por correcta.

Ahora de la misma manera se deben de crear y eliminar tantas preguntas como se quieran hasta que finalicemos el curso pulsando la opción de guaardar curso.

- [Ver ventana crear curso](../diseño/ventanas/VentanaCrearCurso.jpg)

## Ver Mis Cursos: 

Si pulsas esta opción te aparecerá una ventana con los diferentes cursos los cuales tienes disponibles. Un curso esta disponible si se ha creado o importado satisfactoriamente y no se ha eliminado para un creador. Para un estudiante aparecerá si se guarda desde el menu de buscar cursos y no se elimina o se importa.

Si eres un estudiante desde este menu puedes participar en un curso nuevo y deberás elegir una estrategia de aprendizaje de entre las siguientes:

- Secuencial: Aparecerán uno a uno los flashcards correspondientes al curso seleccionado sin que haya ningún aditivo especial. Cuando se responda al ultimo el curso finalizará.

- Repetición: Aparecerán uno a uno los flashcards correspondientes al curso seleccionado pero si fallas alguno se te volverá a preguntar al final del curso.

- Contra Reloj: Aparecerán uno a uno los flashcards correspondientes al curso seleccionado pero tienes 10 segundos para responder a cada uno. Si se acaba el tiempo el curso se terminará dándose este por no superado y se reiniciara automáticamente (no se guarda el progreso).

Al final de un curso se te dará tu puntuación obtenida en este.

Si seleccionas un curso y le das a eliminar podrás eliminar un curso de tu biblioteca. Se pedirá una confirmación antes de eliminarlo.

- [Ver ventana listado cursos estudiante](../diseño/ventanas/VentanaCursosEstudiante.jpg)

- [Ver ventana listado cursos creador](../diseño/ventanas/VentanaCursosCreador.jpg)

- [Ver ventana curso en proceso](../diseño/ventanas/VentanaCurso.jpg)

## Buscar Curso:

En cualquier momento como creador o estudiante puedes buscar cursos existentes. Como creador puedes guardarlos en tu equipo para modificar cursos de otros creadores. Como estudiante puedes guardarlos en tu lista de cursos.

- [Ver ventana buscar curso](../diseño/ventanas/BuscarCurso.jpg)

## Estadísticas:

Aquí podrás consultar una serie de datos relacionados a tu cuenta como el número de cursos superados, tasa de fallos, tiempo de juego, etc.

- [Ver ventana estadisticas estudiante](../diseño/ventanas/EstadisticasEstudiante.jpg)

- [Ver ventana estadisticas creador](../diseño/ventanas/VentanaEstadisticasCreador.jpg)

## Cerrar Sesión:

Al pulsarlo saldrás a la ventana de inicio de sesión.

## Guardado durante un curso:

Durante el curso puedes pulsar el botón guardar y cerrar para dejar el curso a medias y guardar el progreso.
